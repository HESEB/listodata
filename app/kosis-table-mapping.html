입력 순서

1. KOSIS에서 통계표를 열고 기관코드·통계표ID를 확인합니다.
2. 상단의 실제 코드 입력 화면에서 항목코드와 축종 분류코드를 입력합니다.
3. 생성된 JSON을 복사하거나 다운로드해 app/data/config/kosis_table_mapping_template.json에 반영합니다.
4. Actions 실행 후 이 화면에서 ready 여부를 확인합니다.
5. 인증키와 전체 URL은 템플릿이 아니라 GitHub Actions Secret에만 등록합니다.
